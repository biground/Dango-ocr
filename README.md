# Dango-OCR(Windows, Mac)

#### 介绍:
DangoOCR是一款开源的屏幕文字识别工具,通过设置识别范围,自动截屏并调用识别接口实现图片中文字的识别提取。特点：
+ 不必注册复杂的API账号,即下即用
+ 目前支持汉语, 日语, 英语, 韩语的文字识别
+ 针对游戏场景进行了优化, 识别效果可以和百度高精度版媲美
+ 支持windows, mac操作系统
